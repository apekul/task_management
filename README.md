## About

## Demo
    marvelous-longma-a6d022.netlify.app

## Used
    React 16.13.1
    Node v18.15.0
    tailwindcss 3.3.2

## Features
- Add/Update/Remove Projects, tasks, subtask, Columns
- Columns and tasks can be sorted using drag and drop functinalities
